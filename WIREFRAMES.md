## Animation Preferences

- Card hover: Subtle scale (1.02) + shadow increase (200ms ease-out)
- Button press: Background color transition (100ms ease-in-out)
- Modal open: Fade in (300ms ease-out) + scale up (1.05)
- Smooth scrolling: Enabled for anchor links
- Page transitions: Fade (150ms) for smooth navigation
